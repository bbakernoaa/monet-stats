# MONET Stats

A comprehensive statistics and utility library for atmospheric sciences.

This is the statistics and utility functionality spun off from the main MONET repository.

## CI/CD and Code Quality

This project uses a comprehensive CI/CD pipeline with the following quality checks:

- **Testing**: Multi-Python version testing (3.8-3.12) with 95%+ coverage
- **Code Formatting**: Black and Ruff formatting enforcement
- **Linting**: Ruff and Pycodestyle linting
- **Type Checking**: MyPy static type analysis
- **Import Organization**: Isort import sorting
- **Security Scanning**: Safety and Bandit security checks
- **Dependency Review**: Automated dependency vulnerability scanning
- **Documentation**: Automated documentation building
- **Packaging**: Automated package building and validation
